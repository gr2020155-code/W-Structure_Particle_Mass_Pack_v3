## 1. The Concept: Mass as Information State

This repository contains the mathematical proof and prediction tables for the **W-Informational Mass Model**.
Unlike the Standard Model, which treats mass as an empirical parameter coupled to the Higgs field, W-Structure defines mass as a **discrete computational cost** of the particle's topology.

### The Governing Equation
There are no arbitrary constants. The informational mass is determined by integer structural complexity $N$:

$$m_{\text{inf}} = \frac{1}{N}, \qquad N = 2^s \cdot p \cdot q$$

* **$s$:** Spin/Orientation index.
* **$p, q$:** Prime topology factors ($\gcd(p,q)=1$).

**Physical Projection (Calibration to Electron):**
To convert to physical units (GeV), we project relative to the electron's known state:
$$m_{phys} = m_{\text{inf}} \cdot m_e \cdot \frac{N_e}{N}$$
*(The only experimental input is the electron mass).*
